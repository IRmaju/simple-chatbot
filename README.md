## 🔧 Installation

1️⃣ **Clone the Repository**  
```bash
 git clone https://github.com/yourusername/chatbot.git
```

2️⃣ **Navigate to the Directory**  
```bash
 cd chatbot
```

3️⃣ **Install Dependencies**  
```bash
 pip install -r requirements.txt
```

4️⃣ **Run the ChatBot**  
```bash
 python chatbot.py
```
